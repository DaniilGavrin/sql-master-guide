# SQL Master Guide

Добро пожаловать в полное руководство по языку структурированных запросов (SQL)!

## 📚 О проекте

Этот проект создан для того, чтобы помочь разработчикам, аналитикам данных и всем интересующимся освоить SQL от базовых концепций до продвинутых техник оптимизации.

## 🎯 Что вы найдёте здесь

### Основы SQL
- **Введение** — что такое SQL и зачем он нужен
- **SELECT запросы** — основы извлечения данных
- **Фильтрация данных** — использование WHERE для точного поиска
- **Сортировка** — упорядочивание результатов с ORDER BY

### Продвинутые темы
- **JOIN операции** — соединение таблиц различными способами
- **Подзапросы** — мощные вложенные запросы
- **Агрегатные функции** — анализ и группировка данных

### Оптимизация
- **Индексы** — ускорение выполнения запросов
- **Планы выполнения** — понимание работы оптимизатора

## 🚀 Быстрый старт

Начните с раздела [Введение](basics/introduction.md), чтобы понять основы, или перейдите сразу к интересующей вас теме.

## 📖 Пример использования

```sql
-- Простой запрос для получения данных
SELECT 
    first_name,
    last_name,
    email
FROM users
WHERE active = 1
ORDER BY last_name ASC;
```
