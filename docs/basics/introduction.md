# Введение в SQL

## Что такое SQL?

**SQL (Structured Query Language)** — это язык программирования, предназначенный для управления данными в реляционных системах управления базами данных (СУБД).

## История SQL

- **1974** — Дональд Чемберлин и Рэймонд Бойс разработали SEQUEL (предшественник SQL)
- **1986** — SQL стал стандартом ANSI
- **1987** — принят как стандарт ISO
- **Современность** — SQL используется практически во всех СУБД

## Зачем изучать SQL?

### Для разработчиков
- Работа с базами данных в приложениях
- Оптимизация запросов
- Проектирование схем данных

### Для аналитиков данных
- Извлечение данных для анализа
- Агрегация и группировка информации
- Создание отчётов

### Для DevOps инженеров
- Администрирование баз данных
- Мониторинг производительности
- Резервное копирование и восстановление

## Основные концепции

### Реляционная модель данных

Данные организованы в **таблицы**, которые состоят из:
- **Строк (records)** — отдельные записи
- **Столбцов (fields)** — атрибуты данных

### Пример таблицы

| id | first_name | last_name | email | created_at |
|----|------------|-----------|-------|------------|
| 1 | Иван | Петров | ivan@example.com | 2024-01-01 |
| 2 | Анна | Сидорова | anna@example.com | 2024-01-02 |
| 3 | Пётр | Иванов | petr@example.com | 2024-01-03 |

## Синтаксис SQL

SQL состоит из нескольких типов команд:

### DDL (Data Definition Language)
- `CREATE` — создание объектов
- `ALTER` — изменение структуры
- `DROP` — удаление объектов

### DML (Data Manipulation Language)
- `SELECT` — выборка данных
- `INSERT` — добавление записей
- `UPDATE` — обновление записей
- `DELETE` — удаление записей

### DCL (Data Control Language)
- `GRANT` — предоставление прав
- `REVOKE` — отзыв прав
