# Продвинутые темы SQL

Добро пожаловать в раздел продвинутых тем! Здесь вы изучите мощные возможности SQL для работы со сложными запросами, оптимизации и анализа данных.

## 📚 Содержание раздела

- [**JOIN (Объединение таблиц)**](./joins.md) — различные типы соединений таблиц
- [**Агрегатные функции**](./aggregates.md) — GROUP BY, HAVING, COUNT, SUM, AVG, MIN, MAX
- [**Подзапросы**](./subqueries.md) — вложенные запросы и их применение
- [**Оконные функции**](./window-functions.md) — ROW_NUMBER, RANK, DENSE_RANK, LEAD, LAG
- [**CTE (Common Table Expressions)**](./cte.md) — рекурсивные и обычные общие табличные выражения
- [**Индексы и оптимизация**](./indexes.md) — создание индексов, анализ производительности
- [**Транзакции**](./transactions.md) — ACID, COMMIT, ROLLBACK, уровни изоляции
- [**Хранимые процедуры и функции**](./procedures.md) — создание и использование

## 🎯 Что вы узнаете

После изучения этого раздела вы сможете:
- Объединять данные из нескольких таблиц эффективно
- Выполнять сложные аналитические запросы
- Оптимизировать производительность запросов
- Работать с транзакциями для обеспечения целостности данных
- Создавать переиспользуемые компоненты базы данных

## 📊 Уровень сложности

Этот раздел предназначен для пользователей, освоивших основы SQL. Рекомендуется начать с изучения [Основ SQL](../basics/introduction.md), если вы ещё не знакомы с базовыми концепциями.
